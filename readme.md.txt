# 🧠 Python Quiz App

This is a simple **Python command-line quiz app** that asks 5 general knowledge questions, tracks the user’s score, and saves the results to a file.

## 📌 Features

- 5-question multiple-choice style quiz
- Case-insensitive answer checking
- Saves user score, name, and date to a file (`quiz_scores.txt`)
- Easy to modify questions and answers
- Beginner-friendly code with comments
